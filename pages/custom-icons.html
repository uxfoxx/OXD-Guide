Usage Guidelines

Icon Sizing: Default size is 12px for buttons and general system icons. You can also use 14px depending on the icon's optical view and context.

Download Options: Hover over any icon to reveal download options for both SVG and PNG formats.

Customization: The SVG viewport technique allows for better control over icon appearance by cropping unnecessary whitespace while maintaining visual clarity.
